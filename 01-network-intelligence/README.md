# Network Intelligence

Network Intelligence enables investigators to identify hidden relationships, ownership structures and control networks across customers, companies, accounts and counterparties.

## Purpose

Transform isolated customer records into connected investigative intelligence.

## Showcase Focus

- Entity Resolution
- Beneficial Ownership Analysis
- Relationship Discovery
- Network Investigation Workflows
